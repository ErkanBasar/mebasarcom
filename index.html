Projects

Look Who's Talking

Developing long-term engaging chatbots to conduct human-like conversations with humans for two technologically and linguistically diverse domains; smoking cessation and safe sex promotion.

Kon ta bAI?

Developing a large language model that is able to understand and generate text in the low-resourced Papiamentu language, which has about 350.000 native speakers in the world.

FloodTags NEAT

Newspaper Event Analysis Tool is a machine learning based end-to-end information processing pipeline performs event detection and information extraction to analyse news articles and collect information about natural disasters.

Dialect2Keyword

A Natural Language Processing application and a web interface to convert words in Dutch dialects into standard Dutch keywords which can be used as the basis for interoperable searches through various dialect dictionaries.

FloodTags Dashboard

An online dashboard to monitor online media for flood information. The Dashboard shows what people are saying about floods in the media. This enables online monitoring new developments related to floods.

Relevancer

Relevancer aims at finding a balance between complexity of a tweet data set and expert needs from this data. The process can be summarized as creating a need based classifier without sacrificing precision, information, and time.

Lama Events

Lama Events is a calendar application listing events in the near future. The events are detected and selected by a fully automatic procedure in the Dutch Twitter stream.
